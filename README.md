# Clawdbot - 汎用AIエージェント

自分のPCで飼う、対話により自己進化する汎用AIエージェント。

## 概要

Clawdbotは、チャットツール（Discord/Slack/Telegram）からローカルPCを直接操作できるAIエージェントです。npmコマンド一発でインストール完了し、ユーザーの使い方に合わせてボットが勝手に多機能になっていきます。

## 最大の特徴：対話により機能増殖する自己進化性

ユーザーが機能を要求した時、Agent自身がそのためのコード（Skill）を書き、即座に自分自身にインストールして実行します。使いながらユーザーに合わせてボットが勝手に多機能になっていくという画期的な体験が可能です。

## 主な機能

### ブラウザ操作
- 情報収集・フォーム入力の自動化
- スクリーンショット取得

### コンテンツ生成
- スライド/資料自動生成
- 動画編集（FFmpeg連携）

### 音声操作
- 音声認識によるハンズフリー対話（Whisper）
- 音声文字起こし

### サービス連携
- メール送信
- カレンダー操作
- Apple Reminder/Note連携
- Obsidian連携
- Spotify連携
- nanobanana連携
- 電話発信

### 記憶機能
- フック設定でセッションログ保存
- メモリディレクトリに情報蓄積
- 「昨日の続きで」と言えば文脈を引き継げる

### システム管理
- サーバー管理
- コード実行
- プロセス制御

## 実行モード

### フルアクセス（デフォルト）
- プロセスがホストOS上で直接動作
- フル権限
- 注意：ファイル削除等のリスクあり

### ゲスト隔離
- ユーザーはHost Mode
- 他者との会話は隔離実行

### サンドボックス（推奨）
- Gateway全体をDockerコンテナ等のサンドボックスに入れる
- 最安全な実行モード

## 導入方法

```bash
# インストール
npm install -g clawdbot@latest

# セットアップウィザード
clawdbot onboard

# モデル選択 → Botトークン設定 → スキル選択 → チャットで対話開始
```

**セットアップ時間：約10分で動作可能**
