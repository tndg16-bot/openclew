# Base ETH 入金ガイド

Clawdbot（エージェント）を動かすためには、Baseネットワーク上のETH（Ethereum）が必要です。
以下のいずれかの方法で、エージェントのウォレットに入金してください。

**エージェントのウォレットアドレス:**
`0x1f8EbB7f94dE2B50F9fFE81f65C9B68EED7fd0CD`

---

## 方法1: 取引所（CEX）から送金する
**最も簡単で手数料が安い方法です**（Coinbase, Binance, Bybit, OKXなど）。

1. 取引所の出金画面を開く。
2. 通貨: **ETH** を選択。
3. アドレス: 上記のエージェントのアドレスを入力。
4. **ネットワーク: Base (Base Mainnet)** を選択（※超重要！）
   - ⚠️ `ERC20` や `Arbitrum` を選ばないでください。
5. 出金を実行。数分で届きます。

---

## 方法2: Metamask等のウォレットからブリッジする
すでにEthereum Mainnetや他のチェーンにETHを持っている場合。

1. [Base Bridge](https://bridge.base.org/) または [Superbridge](https://superbridge.app/base) にアクセス。
2. 自分のウォレットを接続。
3. `Deposit` を選択。
4. 送金先に「エージェントのアドレス」を指定するか、一度自分のBaseウォレットに送ってからエージェントに転送する。
   - ※自分のウォレットから送る際は、Metamaskのネットワークを **Base** に切り替えてから送金してください。

---

## 入金の目安
- ガス代のみなら: **0.005 ETH** (約$15) 程度で数ヶ月動きます。
- トレードやNFT発行もさせるなら: **0.05 ETH** (約$150) 以上を推奨。

入金が完了したら、Clawdbotに `/wallet balance` コマンドで確認させてください。
